    Inner Join
    

    Syntax:
    
SELECT column1, column2, ...
FROM table1
INNER JOIN table2
ON table1.column = table2.column;


    Ex:
    
mysql> SELECT Students.Name, Courses.CourseName 
FROM Students
INNER JOIN Courses 
ON Students.CourseID = Courses.CourseID;

    Output

    

    Table Before Join:

| Name   | CourseID | CourseName |

| Neha   |   C4     | English   |

    Self Join
    

    Syntax:
    
SELECT A.column1, B.column2
FROM table_name A, table_name B
WHERE A.column = B.column;


    Ex:
    
mysql> SELECT A.Name AS StudentName, B.Name AS MentorName
FROM Students A, Students B
WHERE A.MentorID = B.StudentID;

    Output

    Table Before Join:
    
    
    
+-----------+-----------+-----------+
| StudentID | Name      | MentorID  |
+-----------+-----------+-----------+
| 1         | Lucky     | 2         |
| 2         | Ajay      | 3         |
| 3         | Mahi      | NULL      |
| 4         | Neha      | 2         |
+-----------+-----------+-----------+
4 rows in set (0.00 sec)
    
    

    
    After Applying SELF JOIN:
    
    
    
+--------------+-------------+
| StudentName  | MentorName  |
+--------------+-------------+
| Lucky        | Ajay        |
| Ajay         | Mahi        |
| Neha         | Ajay        |
+--------------+-------------+
3 rows in set (0.00 sec)

    Single-row Subquery
    

    Syntax:
    
SELECT column_name
FROM table_name
WHERE column_name = (SELECT column_name FROM another_table WHERE condition);


    Ex:
    
mysql> SELECT name FROM employees
       WHERE salary = (SELECT MAX(salary) FROM employees);

    Output

    
+------------+
| name       |
+------------+
| John Smith |
+------------+
1 row in set (0.02 sec)

    Query

        



    Multi-row Subquery
    

    Syntax:
    
SELECT column_name
FROM table_name
WHERE column_name IN (SELECT column_name FROM another_table WHERE condition);


    Ex:
    
mysql> SELECT name FROM employees
       WHERE department_id IN (SELECT department_id FROM departments WHERE location = 'New York');

    Output

    
+------------+
| name       |
+------------+
| Jane Doe   |
| Alice Lee  |
+------------+
2 rows in set (0.01 sec)
